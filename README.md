## Wymagane pakiety

By zainstalować wszystkie potrzebne pakiety najlepiej uruchomić poniższy kod. `pacman` sam instaluje brakujące paczki.

```
if(!require(pacman)) install.packages("pacman")
pacman::p_load(xts, lubridate, quantmod, zoo)
```
